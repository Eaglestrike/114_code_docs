# WPILib and VSCode

**VScode** is a popular code editing software. **WPILib** is a robotics library created for FRC. More on that [<u>here</u>](https://docs.wpilib.org/en/stable/docs/software/what-is-wpilib.html)

**WPILib VScode** is VScode with built in WPILib extensions. It is the software we use to write and edit all our code, manage projects, to build code, deploy code on the robot, and more.
